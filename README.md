# หารบิลค่าข้าว (Bill Splitter)
# รายละเอียดโปรแกรม

โปรแกรมหารบิลค่าข้าว (Bill Splitter) เป็นโปรแกรมสำหรับช่วยคำนวณยอดเงินที่แต่ละคนต้องจ่าย เมื่อรับประทานอาหารร่วมกันเป็นกลุ่ม โดยผู้ใช้งานสามารถกรอกข้อมูลยอดรวมค่าอาหาร จำนวนคนที่ร่วมรับประทาน และค่าบริการ (Service Charge) เพื่อให้โปรแกรมคำนวณและแสดงยอดเงินที่แต่ละคนต้องจ่ายโดยอัตโนมัติ

# วิธีการทำงาน
ผู้ใช้กรอก ยอดรวมค่าอาหาร (Total Bill)
ผู้ใช้กรอก จำนวนคน ที่ร่วมรับประทานอาหาร (Number of People)
ผู้ใช้กรอก ค่าบริการ (Service Charge) เป็นเปอร์เซ็นต์ (%)
โปรแกรมคำนวณยอดรวมทั้งหมด (ยอดค่าอาหาร + ค่าบริการ)
โปรแกรมหารยอดรวมด้วยจำนวนคน
โปรแกรมแสดงผล ยอดเงินที่แต่ละคนต้องจ่าย
สูตรการคำนวณ
ยอดรวมค่าบริการ = ยอดรวมค่าอาหาร x (Service Charge / 100)
ยอดรวมทั้งหมด   = ยอดรวมค่าอาหาร + ยอดรวมค่าบริการ
ยอดที่แต่ละคนจ่าย = ยอดรวมทั้งหมด / จำนวนคน
ตัวอย่างการใช้งาน
